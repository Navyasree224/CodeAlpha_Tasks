# CodeAlpha Machine Learning Internship Tasks

This repository contains the Machine Learning projects completed during my CodeAlpha Internship.

## Projects

### ❤️ Task 1: Heart Disease Prediction
- Predicts the likelihood of heart disease using Machine Learning.
- Built with Python, Streamlit, Scikit-Learn, Pandas, and NumPy.

### ✍️ Task 2: AI Handwritten Digit Recognition
- Recognizes handwritten digits using a Convolutional Neural Network (CNN) trained on the MNIST dataset.
- Features an interactive Streamlit dashboard with canvas drawing, image upload, confidence score, probability chart, and prediction history.

## Technologies Used
- Python
- TensorFlow
- Scikit-Learn
- Streamlit
- Pandas
- NumPy
- OpenCV
