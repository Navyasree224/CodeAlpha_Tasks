# ✍️ AI Handwritten Digit Recognition

## Description
This project recognizes handwritten digits using a CNN model trained on the MNIST dataset.

## Features
- Draw digits on canvas
- Upload handwritten images
- AI prediction
- Confidence score
- Probability chart
- Prediction history
- Download prediction history

## Technologies Used
- Python
- TensorFlow
- Streamlit
- OpenCV
- NumPy
- Plotly

## Dataset
MNIST Handwritten Digit Dataset

## Run

```bash
streamlit run app.py
```
